Java-To-Python Converter

This conversion website was written to port Java code to Python, but it also supports Jython2.2 (used in Autoplot, JavaScript, and IDL.

A Java AST is created using the JavaParser library, the AST is then rewritten as similar code. This cannot convert all codes, and a human must review the output!

  • Java to Javascript
  • Java to IDL
